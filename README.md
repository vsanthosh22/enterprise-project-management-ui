# NexusWork – Enterprise Project Management Dashboard

A modern, enterprise-grade project management dashboard inspired by tools like Monday.com. Built using HTML, CSS, Bootstrap, and JavaScript, this UI focuses on productivity, clarity, and scalability for teams.

## 📌 Features

- 📊 Interactive dashboard overview
- ✅ Task management system
- 📈 Progress tracking with visual indicators
- 🧑‍🤝‍🧑 Team collaboration UI
- 🔔 Notifications & activity feed
- 📅 Deadline tracking & upcoming tasks
- 🌙 Dark mode modern interface
- 📱 Fully responsive design

1. Dashboard
2. Tasks

4. Files

7. Settings/Profile

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
